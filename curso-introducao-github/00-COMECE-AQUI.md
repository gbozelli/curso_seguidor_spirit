# Introdução ao GitHub

Curso básico sobre Git e GitHub com foco em usuários Windows.
Este módulo explora controle de versão, colaboração e fluxos profissionais de desenvolvimento.

## O que você vai aprender

- A estrutura de um repositório.
- O que são commits, branches e pull requests.
- Como instalar e configurar Git no Windows.
- Fluxo básico de trabalho (add, commit, push, pull).
- Colaboração através de branches e PRs.
- Boas práticas e padrões de commits.

## Organização do curso

### Materiais de Referência

- `README.md`: Guia completo com todos os conceitos, instalação e fluxo de trabalho.
- `EXERCICIOS.md`: 11 exercícios práticos em 3 níveis (iniciante, intermediário, avançado).
- `DICAS-WINDOWS.md`: Configuração específica para Windows, SSH, estrutura de pastas.
- `CHEAT-SHEET.md`: Referência rápida de comandos para consulta durante o trabalho.
- `EXEMPLOS-COMMITS-PRS.md`: Padrões de mensagens de commit e pull requests.
- `slides.tex`: Apresentação LaTeX para aulas e palestras.
- `.gitignore-template`: Template de .gitignore para diferentes linguagens.

## Como estudar

1. Abra o `README.md` para aprender os conceitos.
2. Siga o exercício do nível correspondente no `EXERCICIOS.md`.
3. Use `CHEAT-SHEET.md` para consultar comandos enquanto trabalha.
4. Veja exemplos práticos em `EXEMPLOS-COMMITS-PRS.md`.
5. Se tiver problema, verifique `DICAS-WINDOWS.md`.

## Conceitos principais

- **Repositório (Repo)**: Pasta de projeto com controle de versão.
- **Commit**: Snapshot do projeto em um determinado momento.
- **Branch**: Linha de desenvolvimento independente.
- **Push**: Enviar commits para o GitHub.
- **Pull**: Baixar commits do GitHub.
- **Pull Request**: Solicitação formal para mesclar código de um branch para outro.

## Percursos recomendados

### Para iniciante (3-4 horas)

1. Leia `README.md` (conceitos básicos).
2. Instale Git e configure conforme `DICAS-WINDOWS.md`.
3. Faça os exercícios de Nível 1 em `EXERCICIOS.md`.
4. Salve `CHEAT-SHEET.md` como referência.

### Para intermediário (4-5 horas)

1. Revise `README.md` (branches e PRs).
2. Faça exercícios de Nível 2 em `EXERCICIOS.md`.
3. Estude padrões em `EXEMPLOS-COMMITS-PRS.md`.
4. Pratique em repositórios pessoais.

### Para avançado (3-4 horas)

1. Faça exercícios de Nível 3 em `EXERCICIOS.md`.
2. Pratique fluxos profissionais com múltiplos branches.
3. Contribua em projetos open-source.

### Para facilitador/professor (8 horas prep)

1. Leia todo o `README.md`.
2. Customize `slides.tex` com seu logo e cores.
3. Compile: `pdflatex slides.tex`.
4. Faça todos os exercícios do `EXERCICIOS.md`.
5. Prepare ambiente com Git instalado e testado.
