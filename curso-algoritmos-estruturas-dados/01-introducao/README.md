# 01-introducao

Introdução a algoritmos e estruturas de dados.
Este tópico explica os conceitos básicos e o raciocínio que sustenta qualquer implementação.

## O que você vai aprender

- O que é um algoritmo.
- O que é uma estrutura de dados.
- Diferença entre dados e operações.
- Como medir eficiência com complexidade.
- Por que usar C para algoritmos e estruturas.

## O que é um algoritmo

Um algoritmo é uma sequência de passos que transforma entradas em saídas.
Ele descreve como resolver um problema de forma clara e finita.

Exemplo simples:

- Entrada: uma lista de números.
- Processamento: ordenar os números.
- Saída: lista ordenada.

Um algoritmo não depende da linguagem, mas da lógica.

## O que é uma estrutura de dados

Uma estrutura de dados organiza informações na memória para que possam ser usadas de forma eficiente.
Alguns exemplos:

- Array: sequência de elementos contíguos.
- Lista ligada: nós encadeados por ponteiros.
- Árvore: hierarquia de nós.
- Grafo: conexões entre vértices.

A escolha correta de estrutura de dados influencia diretamente a velocidade e a simplicidade do algoritmo.

## Conceitos importantes

- Entrada, processamento e saída.
- Estado do programa: valores armazenados em variáveis e estruturas.
- Operações básicas: inserção, remoção, busca, ordenação e travessia.
- Complexidade de tempo: quanto tempo o algoritmo leva em função do tamanho dos dados.
- Complexidade de espaço: quanto de memória o algoritmo usa.
- Abstração: separar o "o quê" do "como".

## Complexidade

As expressões de complexidade mais comuns são:

- O(1): tempo constante.
- O(log n): tempo logarítmico.
- O(n): tempo linear.
- O(n log n): tempo log-linear.
- O(n²): tempo quadrático.

Também falamos em espaço extra usado pelo algoritmo:

- O(1): uso de memória constante.
- O(n): uso de memória proporcional ao tamanho dos dados.

## Por que C?

C é uma linguagem de baixo nível em relação ao código que você escreve, mas ainda é de alto nível para humanos.
Ela permite controlar diretamente alocação de memória e entender melhor o custo das operações.

Os exemplos de implementação são apresentados diretamente neste curso, usando C puro para manter o foco nas estruturas e nos algoritmos.
