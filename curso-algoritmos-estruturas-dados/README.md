# curso-algoritmos-estruturas-dados

Este curso apresenta algoritmos e estruturas de dados fundamentais em C.
O material utiliza implementações locais em C puro para exemplificar cada conceito.

## O que é este curso

- Apresenta o conceito de algoritmo, complexidade e estrutura de dados.
- Mostra como organizar dados em listas, pilhas, filas, árvores e grafos.
- Explica os algoritmos de ordenação, busca e manipulação mais comuns.
- Conecta teoria conceitual com código em C.

## Como o curso está organizado

- `01-introducao`: conceitos de algoritmo, estrutura de dados, entrada/processamento/saída, abstração e complexidade.
- `02-listas`: listas ligadas, nós e operações básicas.
- `03-pilhas`: pilhas estáticas e dinâmicas, uso de LIFO.
- `04-filas`: filas estáticas e dinâmicas, uso de FIFO.
- `05-arvores`: árvores binárias, árvores de busca, B-trees e red-black trees.
- `06-grafos`: representação de grafos, conexão e busca.
- `07-ordenacao`: bubble, insertion, selection, merge, quick e heap sort.
- `08-busca`: busca linear, busca em árvore e busca binária em árvore.
- `09-hashing-e-priority-queues`: hashing e filas de prioridade.

## Como usar este material

1. Abra o `README.md` do tópico antes de executar qualquer código.
2. Leia a explicação do algoritmo ou estrutura de dados.
3. Abra os arquivos desta pasta para ver o código completo.
4. Compile e execute o exemplo em C.
5. Experimente modificar dados e observe as diferenças de comportamento.

## Referências gerais

- "Introduction to Algorithms" - Cormen, Leiserson, Rivest, Stein.
- "Algorithms" - Robert Sedgewick.
- "The C Programming Language" - Kernighan e Ritchie.

## Observação

Este curso foi escrito para complementar o treinamento de novos integrantes, unindo explicações didáticas e código que pode ser estudado diretamente. É um tópico mais avançado de programação.
