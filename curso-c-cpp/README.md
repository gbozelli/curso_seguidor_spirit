# curso-c-cpp

Este módulo é um curso básico de C e C++ para quem quer aprender a programar em linguagens de sistema.
Ele começa pelo programa mais simples e avança para tipos, vetores, funções e orientação a objetos.

## O que você vai aprender

- Estrutura de um programa em C.
- Tipos de dados básicos e variáveis.
- Como usar arrays e strings.
- Como dividir o código em funções.
- Como usar `struct` e conceitos iniciais de orientação a objetos em C++.

## Organização do curso

- `01-introducao`: primeiro programa, compilação e estrutura básica.
- `02-tipos-e-variaveis`: tipos primitivos, variáveis e constantes.
- `03-arrays-e-strings`: vetores, uso de arrays de caracteres e `std::string`.
- `04-funcoes-e-estruturas`: funções, parâmetros, retorno e `struct`.
- `05-estruturas-de-controle`: tipos otimizados, condicionais `if/else` e laços `for`/`while`.
- `06-orientacao-a-objetos`: classes, objetos, construtores e herança simples.
- `07-classes-avancadas`: encapsulamento, herança, polimorfismo e projeto de classes.

## Como estudar este módulo

1. Abra o `README.md` do tópico escolhido.
2. Leia a explicação antes de executar o código.
3. Compile o exemplo no terminal.
4. Modifique valores e observe o resultado.
5. Faça os exercícios sugeridos no final de cada tópico.

## Como compilar

- Para C:
```bash
gcc arquivo.c -o arquivo
```
- Para C++:
```bash
g++ arquivo.cpp -o arquivo
```
- Para executar:
```bash
./arquivo
```

## Pontos importantes

- Em C, a função `main()` é o ponto de entrada do programa.
- Em C++, existe toda uma biblioteca padrão adicional.
- Mesmo em projetos C++, a base de C continua sendo útil.
- Escrever código limpo e comentado facilita o entendimento.

## Referências

- Livro: "The C Programming Language" de Brian Kernighan e Dennis Ritchie.
- Livro: "A Tour of C++" de Bjarne Stroustrup.

## Observação

Este material foi planejado para iniciantes com foco em raciocínio técnico e clareza.
Não é preciso saber todo o conteúdo de uma vez; avance gradualmente e pratique cada exemplo.
